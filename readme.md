## react template
```
yarn start // 启动项目

yarn fix // 根据lint修复代码格式

yarn build // 打包

yarn test // 测试

git remote set-url origin @REPOSITORY // 更改项目git地址
```

#### 目录
```
::
    | -- .vscode  vscode配置
    | -- public   html、favicon
    | -- src    项目代码
        | -- app 根组件
        | -- assets 静态资源
            | -- styles
        | -- pages  页面
        | -- core   可复用的业务逻辑代码
        | -- shared 可复用的与业务逻辑无关的代码
        | -- pages 页面
    | -- typings    ts-node自定义type的目录
    | -- webpack    webpack脚本

```
